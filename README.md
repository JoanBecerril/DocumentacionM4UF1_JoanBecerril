### Estilos de letra
Para definir un estilo a un texto tenemos todos estos tipos:
```
***Estilo***                        Negrita y Cursiva
**Estilo**                          Negrita
*Estilo*                            Cursiva
**Estilo_Estilo_Estilo**            Negrita Cursiva Negrita
>Estilo                             Anotacion
~~Estilo~~                          Tachar
```
